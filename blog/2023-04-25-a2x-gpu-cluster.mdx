A new GPU cluster is now available at the Institute for Artificial Intelligence and Autonomous Systems (A2S).
This new cluster can be used for rapid prototyping and development of machine learning and robotic applications.

There are five nodes available, of which there are up to three Nvidia graphic cards installed.
A ready-to-use machine learning environemnt is available for all users to make it 
easy for users to get started quickly.
